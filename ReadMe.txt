-----------------------------------------
INTER GALACTIC TRADE APPLICATION
-----------------------------------------

Steps to run the project ->

1) Import Project folder into Eclipse by selecting the option Import -> 'Existing projects into workspace'
2) Build & Compile the project.
3) Run the test cases inside the test folder.
4) Run the main() inside galaxy.application.InterGalacticTrade class.
5) User can fetch inputs from the file provided in the classpath (Input.txt) or 
   enter inputs manually. 

This application is developed using JDK 1.6
